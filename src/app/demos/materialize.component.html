Material Design

Diese Starter APP basiert auf Material Design. Ursprünglich entwickelt und ausgearbeitet von Google findet man die Prinzipien des Material Design immer öfter bei modernen Anwendungen.

Materialize CSS

In diesen Angular 2 Starter ist das CSS Framework Materialize CSS integriert. Dieses hält sich strikt an die Designgrundlagen des Material Design und bietet allerlei Funktionalität für die optimale Benutzererfahrung.

Um alle Funktionalitäten des Framework nutzen zu können wird die angular2-materialize Library genutzt.
